# 应急屏语义地图重设计

## 目标

基于真实徐闻港地理信息，重新设计应急屏语义地图，解决两个核心问题：
1. 不符合真实徐闻港口道路情况
2. 地图内容集中在中间，空间利用率低

## 方案：垂直布局 - 北陆南海

保留 SVG 语义化表达风格，参考真实地理布局调整海陆比例和道路走向。

### ViewBox 空间划分 (1200x680)

- y=0-400: 陆地区域(59%) - 县城、高速网络、进港大道、港口设施
- y=400-420: 海岸线过渡带(3%) - 渐变色弧线
- y=420-680: 海域区域(38%) - 琼州海峡、台风、航线、海口
