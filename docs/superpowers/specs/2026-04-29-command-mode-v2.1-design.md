> 版本：V1.0 | 日期：2026-04-29
> 基于用户测试反馈，针对指挥模式的 8 项改进

---

## 1. 地图图层筛选

**位置：** 地图左上角

**筛选按钮组：**
```
[全部] [拥堵路段] [执行人员] [设备点位] [分流路线] [特殊车辆]
```

**默认状态：** 只显示"拥堵路段 + 执行人员"
**交互：** 点击切换，支持多选

---

## 2. 地图元素与策略执行联动

**核心逻辑：** 地图从"态势图"升级为"策略执行图"，元素随执行进度分阶段展示。

### 未执行前
- 地图只显示问题态势（拥堵路段、排队长度）
- 推荐路线用淡色虚线提示，不强展示

### 执行确认后（按时间顺序）

| 时间 | 展示内容 | 视觉表现 |
|------|---------|---------|
| 立即（0s） | 路线变化：分流路线/应急车道/停车区 | 淡虚线 → 亮色动态流线 |
| 30s 内 | 人员变化：交警出发、拖车调度 | 人员图标开始移动动画 |
| 1-3min | 设备变化：诱导屏更新、信号灯切换 | 设备图标状态变更 + 闪烁提示 |
| 5min 后 | 结果变化：拥堵热区变化、排队缩短 | 热力颜色渐变、排队长度动画缩短 |

### 执行状态视觉编码

| 状态 | 视觉表现 |
|------|---------|
| 待执行 | 淡色虚线 |
| 执行中 | 亮色动态流线（青色流动） |
| 已生效 | 稳定高亮 + 指标变化标注 |
| 异常/偏差 | 红色闪烁 + 偏差标注 |

## 3. 视频通话延迟触发

**当前问题：** 进入指挥模式立即弹出视频通话，打断领导先看态势

**改为：**
- 进入指挥模式时不自动弹出视频
- 策略执行确认后延迟 5 秒触发
- 自动选择与当前策略最相关的视频源

**正确的指挥节奏：**
```
进入指挥模式 → 看态势 → 选策略 → 点执行 → 5秒后视频接入 → 执行协同
```

---

## 4. 视频源与策略关联

**逻辑：** 根据执行的策略类型，自动切换到最相关的视频源

| 策略类型 | 关联视频源 |
|---------|-----------|
| S-02/S-03 分流 | 分流路口摄像头 + 无人机俯拍 |
| S-01/S-13 车道借用 | 进港大道摄像头 |
| S-07 事故处置 | 事故点摄像头 + 现场对讲 |
| S-04 信号灯优化 | 关键路口摄像头 |
| S-08 临时停车区 | 停车区摄像头 |
| S-14 交警增援 | 部署点位摄像头 |

**视频面板状态：**
- 未执行前：小窗收起或弱展示
- 执行后：自动提升视频权重，弹出相关视频源

## 6. 自定义策略

**入口：** 右侧面板底部"自定义策略"按钮

**点击后弹出面板，两种模式：**

### Tab 1：从策略库选择
- 显示 15 个标准策略列表
- 每个策略显示名称 + 简要描述
- 可多选组合
- 底部"确认执行"按钮

### Tab 2：手动新增
- 策略名称（必填）
- 执行动作描述（必填）
- 预期效果说明（选填）
- 底部"添加并执行"按钮

## 7. 资源面板按执行状态切换

| 执行状态 | 面板标题 | 显示内容 |
|---------|---------|---------|
| 未执行 | 当前可用资源 | 在岗交警 X 人、可用拖车 X 辆、诱导屏 X 块 |
| 已选未执行 | 策略所需资源 | 需要 vs 可用对比、是否满足、到位时间 |
| 执行中 | 执行资源状态 | 谁在哪、到位了没、物资消耗 |
| 执行完毕 | 隐藏或显示释放状态 | 资源释放情况 |

---

## 8. 去掉右上角返回按钮

- 移除右上角常驻的"返回总览"按钮
- 改为：执行完毕阶段，在右侧面板提供"返回总览模式"按钮

---

## 策略卡片信息升级

每个策略卡片必须包含：

1. **推荐理由**（1-2 句话，基于归因结果）
2. **历史依据**（执行次数 / 成功率 / 匹配度 / 置信度）
3. **预期效果**（资源到位时间 + 策略生效时间 + 各检查点预期值）
4. **资源需求摘要**（需要什么 / 是否满足）
5. **执行按钮**
