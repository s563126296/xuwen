# 徐闻县智慧交通大屏系统

## 项目概述

为徐闻县（中国大陆唯一通往海南的陆路口岸）开发智慧交通大屏，支持总览、指挥、港口、分析 4 种模式。技术栈：React 18 + TypeScript + Vite + Zustand + Recharts + 高德地图。

## 当前状态

- 总览模式 AI 增强：虚拟人助手"小语" + AI 预测指标强化 + AI 摘要面板
- 指挥模式应急合并：场景感知（congestion/emergency）+ 应急面板 + 策略时间线

## 关键约束

- 深色主题：#0A0F19 背景，#00D0E9 青色主题
- 不用 emoji 做 UI 图标，统一用 Lucide React
- 坐标系 GCJ-02（高德），部署在政务内网
- **不要过度设计**：功能点要服务于真实用户场景，不堆砌
- **优先 AI 价值体现**：每个 AI 功能都要有明确的业务价值和用户感知

## 已知问题

3. 策略推荐缺少推荐理由和适用场景说明
