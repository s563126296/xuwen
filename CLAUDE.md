## 业务要点（建设方案核心）

### 拥堵真实成因（重要！）
预约系统已从源头控制装船车辆总量，不会超过港口消化能力。拥堵真因是**道路上车流过载**：
- **早到车流**：预约规则只允许提前2小时进港，但很多车提前更久就到了
- **送人车流**：单船载客~165人 >> 载车~65辆，节假日纯坐船人通过出租车/私家车/摩托车送行
- **过路车流**：本地通勤、过路车辆叠加

### 治理机制
- **预约制**：所有车辆和人员进港必须预约
- **四级响应**：常态→预警通知→道路分流→全面管控
- **核心指标**：早到车流压力 + 送人车流压力 + 道路通行余量 + 气象影响系数
- **送客通道**：进港大道最左侧1股车道专供送人车（出租车/私家车/摩托车）；主车道2-3股供已到时段进港车排队
- **通知双轨**：对车辆预约用户（早到提示去停车场）+ 对人员预约用户（出行建议）
- **无人机**：2台+飞手，预警/分流/管控三阶段均可使用（视频+统计+喊话）
- **应急融合通信**：应急局规划建设，本系统预留集成接口
- **应急物资保障**：滞留时发放盒饭/饮用水，疏散驾驶员和特殊车辆

### 关键数据
- 日均进港车辆 ~8,000（平日）/ 5万（春运峰值）
- 港口日发船 ~107 班次（徐闻港86 + 海安新港21）
- 单船平均载车~65辆 / 载客~165人
- 主要拥堵路段：进港大道14.8km + 徐海路4.2km + 环半岛公路7.3km
