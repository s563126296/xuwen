## 1. 色彩系统

### 1.1 主色（Primary Colors）

| 用途 | 色值 | CSS 变量 | 使用场景 |
|------|------|----------|---------|
| 主色 | `#4DA6FF` | `--color-primary` | 主要按钮、链接、高亮元素 |
| 主色发光 | `rgba(77, 166, 255, 0.5)` | `--color-primary-glow` | 数据发光效果 |

**对比度：** 在深色背景 `#060d1a` 上对比度为 8.2:1（WCAG AAA）

**使用示例：**
```css
.primary-button {
  background: var(--color-primary);
  color: #fff;
}

.data-highlight {
  color: var(--color-primary);
  text-shadow: var(--glow-primary);
}
```

### 1.2 辅助色（Secondary Colors）

| 用途 | 色值 | CSS 变量 | 使用场景 |
|------|------|----------|---------|
| 次要色 | `#A78BFA` | `--color-secondary` | 次要强调、紫色主题（仅限特定场景） |
| 强调色 | `#F0B429` | `--color-accent` | 警告级别、重要数据 |
| 强调色发光 | `rgba(240, 180, 41, 0.4)` | `--color-accent-glow` | 强调数据发光 |

**注意：** `--color-secondary` 紫色仅用于特殊标识，不作为大面积主题色使用。

### 1.3 语义色（Semantic Colors）

| 语义 | 色值 | CSS 变量 | 使用场景 | 对比度 |
|------|------|----------|---------|--------|
| 成功/正常 | `#34D399` | `--color-success` | 正常状态、成功提示 | 7.8:1 (AAA) |
| 警告/注意 | `#FBBF24` | `--color-warning` | 警告提示、中等风险 | 10.2:1 (AAA) |
| 危险/错误（软） | `#FB923C` | `--color-danger-soft` | 中度危险 | 6.5:1 (AA) |
| 危险/错误 | `#F87171` | `--color-danger` | 严重错误、高风险 | 5.8:1 (AA) |

**使用规则：**
- 绿色：设备在线、指标正常、操作成功
- 黄色：需要关注、中等警告
- 橙色：较严重警告、需要处理
- 红色：严重错误、紧急情况

### 1.4 背景色（Background Colors）

| 层级 | 色值 | CSS 变量 | 使用场景 |
|------|------|----------|---------|
| 主背景（虚空） | `#060D1A` | `--bg-void` | 页面主背景 |
| 深层背景 | `#0A1628` | `--bg-deep` | 次级背景层 |
| 表面背景 | `#0F1F3A` | `--bg-surface` | 卡片内部背景 |

**层级关系：** void（最深） < deep < surface

### 1.5 文字色（Text Colors）

| 层级 | 色值 | CSS 变量 | 使用场景 | 对比度 |
|------|------|----------|---------|--------|
| 主文字 | `rgba(255, 255, 255, 0.92)` | `--text-primary` | 标题、重要文字 | 13.8:1 (AAA) |
| 次文字 | `rgba(180, 200, 220, 0.65)` | `--text-secondary` | 说明文字、标签 | 7.2:1 (AAA) |
| 辅助文字 | `rgba(140, 160, 180, 0.45)` | `--text-tertiary` | 次要信息、禁用状态 | 4.8:1 (AA) |
| 数据文字 | `rgba(255, 255, 255, 0.88)` | `--text-data` | 数字、数据展示 | 13.2:1 (AAA) |
| 高亮文字 | `rgba(255, 255, 255, 0.98)` | `--text-highlight` | 强调、高亮 | 14.5:1 (AAA) |

**使用规则：**
- 标题、关键信息：`--text-primary` 或 `--text-highlight`
- 说明文字、标签：`--text-secondary`
- 辅助信息、次要内容：`--text-tertiary`
- 数字、数据：`--text-data` + 数据字体

### 1.6 边框与分割线

| 用途 | 色值 | CSS 变量 |
|------|------|----------|
| 玻璃边框（默认） | `rgba(100, 180, 255, 0.08)` | `--glass-border` |
| 玻璃边框（hover） | `rgba(100, 180, 255, 0.18)` | `--glass-border-hover` |
| 分割线 | `rgba(180, 200, 220, 0.08)` | `--divider` |
| 分割线（微弱） | `rgba(180, 200, 220, 0.04)` | `--divider-subtle` |
| 分割线（强调） | `rgba(240, 180, 41, 0.12)` | `--divider-accent` |

### 1.7 发光效果（Glow Effects）

| 类型 | 色值 | CSS 变量 |
|------|------|----------|
| 主色发光 | `0 0 12px rgba(77, 166, 255, 0.35)` | `--glow-primary` |
| 强调色发光 | `0 0 12px rgba(240, 180, 41, 0.3)` | `--glow-accent` |
| 危险色发光 | `0 0 10px rgba(248, 113, 113, 0.3)` | `--glow-danger` |
| 成功色发光 | `0 0 10px rgba(52, 211, 153, 0.3)` | `--glow-success` |

**使用场景：** 数据数字、状态指示点、重要按钮

## 2. 字体系统

### 2.1 字体家族

| 用途 | 字体 | CSS 变量 | 使用场景 |
|------|------|----------|---------|
| 显示字体 | Orbitron | `--font-display` | 大标题、Logo、特殊标识 |
| 标签字体 | Rajdhani | `--font-label` | 标签、小标题 |
| 正文字体 | Noto Sans SC | `--font-body` | 正文、说明文字 |
| 数据字体 | JetBrains Mono | `--font-data` | 数字、数据、代码 |

**Fallback 链：**
```css
--font-body: 'Noto Sans SC', -apple-system, BlinkMacSystemFont, sans-serif;
--font-data: 'JetBrains Mono', 'DIN', 'DIN Alternate', 'Roboto Mono', monospace;
```

| H3 | 19px | 1.3 | 卡片标题 |
| H4 | 16px | 1.4 | 小节标题 |
| H5 | 14px | 1.4 | 次级标题 |
| H6 | 13px | 1.4 | 最小标题 |

## 5. 图表样式规范

### 5.1 折线图（Line Chart）

**线条样式：**
- 线条粗细：2px
- 主线颜色：`#4DA6FF`（主色）
- 辅助线颜色：`#F0B429`（强调色）
- 数据点：圆形，半径 4px
- 数据点 hover：半径 6px + 发光效果

**网格线：**
- 颜色：`rgba(180, 200, 220, 0.08)`
- 粗细：1px
- 样式：虚线（dash: [4, 4]）

**坐标轴：**
- 轴线颜色：`rgba(180, 200, 220, 0.12)`
- 轴线粗细：1px
- 标签颜色：`--text-tertiary`
- 标签字号：11px

**示例配置（ECharts）：**
```javascript
{
  color: ['#4DA6FF', '#F0B429', '#34D399'],
  lineStyle: { width: 2 },
  itemStyle: {
    borderWidth: 2,
    borderColor: '#4DA6FF',
    shadowBlur: 4,
    shadowColor: 'rgba(77, 166, 255, 0.3)'
  },
  grid: {
    borderColor: 'rgba(180, 200, 220, 0.08)',
    splitLine: {
      lineStyle: {
        color: 'rgba(180, 200, 220, 0.08)',
        type: 'dashed'
      }
    }
  }
}
```

### 5.2 柱状图（Bar Chart）

**柱子样式：**
- 柱子颜色：渐变（从 `#4DA6FF` 到 `rgba(77, 166, 255, 0.3)`）
- 柱子圆角：顶部 4px
- 柱子间距：20%
- 柱子最小宽度：8px

**示例配置：**
```javascript
{
  itemStyle: {
    borderRadius: [4, 4, 0, 0],
    color: {
      type: 'linear',
      x: 0, y: 0, x2: 0, y2: 1,
      colorStops: [
        { offset: 0, color: '#4DA6FF' },
        { offset: 1, color: 'rgba(77, 166, 255, 0.3)' }
      ]
    }
  },
  barCategoryGap: '20%'
}
```

### 5.3 饼图/环图（Pie/Donut Chart）

**颜色序列：**
```javascript
['#4DA6FF', '#34D399', '#F0B429', '#FB923C', '#F87171', '#A78BFA']
```

**样式：**
- 描边：2px，颜色 `#060D1A`
- 标签线：1px，颜色 `rgba(180, 200, 220, 0.3)`
- 标签字号：12px
- hover 偏移：10px

### 5.4 地图样式

**底图主题：** 深色

**标注样式：**
- 点标注：圆形，半径 6px，颜色 `#4DA6FF`，发光效果
- 线标注：粗细 2px，颜色 `#4DA6FF`，虚线动画
- 面标注：填充 `rgba(77, 166, 255, 0.15)`，边框 `#4DA6FF`

**热力图配色：**

## 6. 交互规范

### 6.1 hover 效果

**卡片 hover：**
```css
.card:hover {
  border-color: rgba(100, 180, 255, 0.18);
  transform: translateY(-1px);
  box-shadow: 0 12px 40px rgba(0, 0, 0, 0.5),
              0 0 10px rgba(100, 180, 255, 0.15);
  transition: all 200ms cubic-bezier(0.16, 1, 0.3, 1);
}
```

**按钮 hover：**
```css
.button:hover {
  background: linear-gradient(135deg, 
    rgba(77, 166, 255, 0.3) 0%, 
    rgba(77, 166, 255, 0.2) 100%);
  box-shadow: 0 4px 16px rgba(77, 166, 255, 0.2);
  transform: translateY(-1px);
}
```

### 6.2 transition 规范

| 速度 | 时长 | CSS 变量 | 使用场景 |
|------|------|----------|---------|
| 快速 | 200ms | `--duration-fast` | hover、focus |
| 正常 | 400ms | `--duration-normal` | 展开/收起 |
| 慢速 | 600ms | `--duration-slow` | 页面切换 |

**缓动函数：**
- 标准缓出：`cubic-bezier(0.16, 1, 0.3, 1)` (`--ease-out`)
- 弹性缓动：`cubic-bezier(0.34, 1.56, 0.64, 1)` (`--ease-spring`)

### 6.3 动画

**淡入淡出：**
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

**滑入：**
```css
@keyframes slideInRight {
  from {
    opacity: 0;
    transform: translateX(-20px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}
```

## 10. 自适应规则

### 10.1 断点

| 断点 | 宽度 | 说明 |
|------|------|------|
| 大屏 | ≥ 1920px | 标准大屏 |
| 中屏 | 1440px - 1919px | 笔记本 |
| 小屏 | < 1440px | 不推荐 |

### 10.2 自适应策略

**大屏（≥ 1920px）：**
- 使用标准布局
- 所有面板正常显示

**中屏（1440px - 1919px）：**
- 侧栏宽度缩小到 280px
- 字号保持不变
- 间距适当缩小

**小屏（< 1440px）：**
- 不推荐使用
- 如需支持，建议隐藏部分次要面板
