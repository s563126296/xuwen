# 应急模式 — 临时停车区选址分析

> 版本：V1.0 | 日期：2026-04-13
> 说明：基于系统截图路网信息、需求文档、徐闻港地理位置综合分析

---

## 一、分析依据

### 1.1 已知路网信息（来自系统截图+需求文档）

```text
徐闻城区 ←→ 徐闻港 核心路网：

  徐闻城区
      │
      ├── 城区路口（城区进入进港大道的汇入点）
      │
      ├── 华四村（X699县道与进港大道交叉口）
      │         ↘ X699县道（通往北部）
      │
      ├── 进港大道（核心通道，单向3车道，正在拓宽）
      │         │
      │         ├── 中段（高速入口路口）
      │         │         ↗ 连接高速公路
      │         │
      │         ├── 十亩村路口
      │         │
      │         └── S376省道交叉口
      │                   ↘ S376省道（备选分流路线）
      │
      └── 港口大道 → 徐闻港

  G207国道（纵向穿过，备选分流路线）
  S376省道（连接进港大道和港口大道的备选路线）
```

### 1.2 选址原则

1. **距离进港大道 < 3公里** — 太远司机不愿意去，太近会加剧进港大道拥堵
2. **面积 > 5,000㎡** — 至少能容纳150辆车才有实际意义
3. **有基本的进出通道** — 能从主干道进出，不需要穿越村庄或窄路
4. **不影响其他重要交通节点** — 不能堵住路口或占用应急通道
5. **地面条件** — 硬化地面优先，其次碎石/压实土地，避免软土泥地（台风暴雨会泥泞）
6. **优先利用现有设施** — 已有停车场/广场/废弃场地优先

## 二、推荐选址方案

### P-1：港口周边待开发用地（距港口约1-2km）

```text
位置：进港大道末段至港口之间的道路两侧
理由：
  - 徐闻港是新建港口（2022年投入使用），周边尚有大量待开发用地
  - 港口规划区域内通常预留有大面积硬化空地
  - 距离港口最近，复航后车辆可最快进入排队上船
  
预估参数：
  面积：约15,000-20,000㎡（港口周边空地通常较大）
  地面：部分硬化（港口建设期遗留）+ 部分碎石
  容量：约400-550辆（按货车60%+小车40%计算）
  进出通道：直接从进港大道/港口大道进出
  
优势：
  ✅ 距港口最近，复航放行效率最高
  ✅ 面积大，容量充足
  ✅ 进出便捷，不需要绕行
  
劣势：
  ⚠️ 需确认用地权属（可能属于港口管理方/土地储备中心）
  ⚠️ 台风期间可能受风力影响较大（靠近海岸）
  
适用场景：所有应急场景的首选停车区
冷链车专区：靠近出口位置划定，面积约2,000㎡（约55辆冷链车）
```

### P-2：S376省道与进港大道交叉口附近空地（距港口约3-4km）

```text
位置：S376省道与进港大道交汇区域的道路两侧
理由：
  - 两条主干道交汇处，交通可达性好
  - 该区域为农村腹地，道路两侧有农田和空地
  - 是分流路线的关键节点，停车区设在这里可配合S-02分流策略
  
预估参数：
  面积：约8,000-12,000㎡
  地面：碎石路面/压实土地（需简单硬化处理）
  容量：约220-330辆
  进出通道：从S376省道或进港大道进出
  
优势：
  ✅ 位于分流节点，可同时服务进港大道和S376两条线路的车辆
  ✅ 距离城区适中，不会加剧城区拥堵
  ✅ 周边空地较多，有扩展空间
  
劣势：
  ⚠️ 地面可能需要简单硬化处理
  ⚠️ 暴雨期间排水条件需评估
  
适用场景：滞留500-1500辆时启用（配合P-1）
```

### P-3：城区路口外围/高速入口附近（距港口约5-6km）

```text
位置：高速公路入口路口附近区域
理由：
  - 高速入口通常有较宽的道路和辅道空间
  - 可拦截从高速下来的进港车辆，直接引导进入停车区
  - 防止车辆继续驶入已拥堵的进港大道
  
预估参数：
  面积：约6,000-8,000㎡
  地面：部分硬化（高速辅道区域）
  容量：约170-220辆
  进出通道：高速出口辅道 + G207国道
  
优势：
  ✅ 截流作用——从源头拦截车辆，减少进港大道压力
  ✅ 高速出口通常道路条件较好
  ✅ 可配合S-06限流策略使用
  
劣势：
  ⚠️ 距港口较远，复航后放行需要较长时间
  ⚠️ 需要与高速公路管理方协调用地
  
适用场景：滞留>1500辆时启用（配合P-1+P-2）
```

### P-4：徐闻城区大型停车场/广场（备选方案）

```text
位置：徐闻县城内现有的公共停车场、体育场停车区、商业广场等
理由：
  - 利用现有设施，无需额外建设
  - 城区内生活配套完善（司机可就近吃饭、休息）
  
预估参数：
  面积：分散，总计约5,000-10,000㎡
  地面：硬化
  容量：约150-300辆（分散在多个点位）
  
优势：
  ✅ 现有设施，即开即用
  ✅ 生活配套好，司机体验较好
  ✅ 适合小型客车停放
  
劣势：
  ⚠️ 容量有限，分散管理困难
  ⚠️ 大型货车/冷链车不适合进入城区
  ⚠️ 可能影响城区居民正常停车需求
  
适用场景：仅用于小型客车分流，不适合货车
```

## 三、启用策略

### 3.1 分级启用规则

```text
滞留规模            启用方案                    总容量
< 500辆            P-1                         ~500辆
500-1000辆         P-1 + P-2                   ~800辆
1000-1500辆        P-1 + P-2 + P-4(小客车)     ~1100辆
> 1500辆           P-1 + P-2 + P-3 + P-4       ~1300辆
> 2000辆           全部启用 + 道路两侧临时停靠   ~1800辆+
```

### 3.2 车型分配

```text
P-1（港口周边）：
  优先：冷链车（靠近出口，便于优先放行）
  其次：危化品车（需隔离区域）
  剩余：普通货车

P-2（S376交叉口）：
  主要：普通货车
  次要：大型客车

P-3（高速入口）：
  主要：从高速下来的截流车辆（各类型）

P-4（城区）：
  仅限：小型客车
```

### 3.3 台风场景的特殊考虑

```text
台风期间停车区使用需注意：

1. 风力约束
   P-1（靠海岸）：风力>10级时，大型货车有侧翻风险
   → 建议货车停放在P-2（内陆，风力较小）
   → P-1仅保留冷链车（车身重，侧翻风险低）

2. 排水条件
   暴雨期间地面积水会导致车辆无法移动
   → P-2/P-3如为非硬化地面，需提前铺设临时排水沟
   → 备选方案：在进港大道路面直接划定临时停车段

3. 供电保障
   冷链车长时间滞留需要持续供电/供油
   → P-1冷链车专区应靠近有电源的区域
   → 备选：调配移动发电车为冷链车供电

4. 照明条件
   夜间停车区需要照明（安全+管理需要）
   → 优先使用有路灯的区域
   → 备选：调配移动照明车
```

## 四、建设建议

### 4.1 短期（零投入，利用现有资源）

- 与港口管理方协商P-1用地的应急征用协议
- 摸排P-2/P-3的具体可用面积，现场拍照建档
- 将P-4（城区停车场）纳入应急征用预案
- 在大屏系统中标注各停车区位置（即使未启用）

### 4.2 中期（小额投入，约10-20万）

- P-2地面简单硬化处理（碎石铺设+压实），约5-10万
- 各停车区设置临时标识牌和进出指引
- 采购移动照明设备2-3台，约3-5万
- 采购移动发电车1台（冷链车供电），约5-10万

### 4.3 长期（纳入基建规划）

- 在进港大道沿线规划正式的货车待港停车场
- 参考其他港口城市的做法（如连云港、日照港均有专用待港停车场）
- 面积建议：30,000㎡以上（可容纳800+辆货车）
- 配套：供水供电、卫生间、便利店、加油站
