# 01 总览模式（Overview Mode）

## 定位

日常值班默认模式，全局态势感知，无需干预时的常态展示。

## 当前状态

现有系统即为总览模式，已具备基础功能：
- 左栏：感知设备统计（119台，97%在线）+ 徐闻出行 + 交通违法事件
- 中央：GIS地图（路网 + 卡口点位 + 路况着色）+ 24小时车流量趋势
- 右栏：双港总览（进港/出港拥堵指数）+ 拥堵预测 + 堵点预警
- 顶部：预警信息滚动 + 港口切换 + 系统时间

## 升级方向

1. GIS地图升级为高德真实底图，叠加实时路况热力
2. 右栏新增"海峡通行指数"核心指标
3. 预警信息增加智能分级（黄/橙/红）
4. 增加模式切换导航入口

## 优先级

P0 — 在现有基础上升级，工作量最小
