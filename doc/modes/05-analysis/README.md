# 05 分析模式（Analysis Mode）

## 定位

历史回溯、模拟推演、领导汇报。让大屏从"实时工具"变成"分析工具"。

## 触发方式

- 手动切换

## 核心功能规划

1. 时间轴回放（拖动时间轴回放任意时段全域态势）
2. 场景对比（今年vs去年、工作日vs节假日、春运vs平日）
3. 模拟推演（输入参数如"流量+20%"，预测交通影响）
4. AI自动报告（每日/每周交通运行报告自动生成）
5. 关键事件时间轴（标记历史重大拥堵/台风事件节点）

## 数据依赖

- 历史数据积累（至少3-6个月）
- 各数据源历史归档

## 优先级

P2 — 需要历史数据积累，最后实施
