# 02 港口模式（Port Mode）

## 定位

聚焦港口运营与海峡通行，掌握运力调度全貌。

## 触发方式

- 手动切换
- 大船靠港时（AIS数据）右下角提示建议切换

## 核心功能规划

1. 海峡通行指数（综合评估海峡通行能力 0-100）
2. 船舶动态（AIS实时位置、航速、航向、预计到港时间）
3. 港口运力详情（可运船舶、可运车辆、当前装载率）
4. 班次时刻表（下一班次、剩余车位、预计发船时间）
5. 待舶队列预测（当前排队数 → 预计等待时长）
6. 两港对比分析（徐闻港 vs 海安新港运力对比）

## 数据依赖

- AIS船舶数据（公开数据 + 船讯网API）
- 港口闸机实时数据（需推动港口方对接）
- 海洋气象数据（风力、浪高、能见度）

## 优先级

P1 — 依赖AIS和港口数据接入
