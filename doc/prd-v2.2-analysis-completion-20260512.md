## 2. 产品概述

### 2.1 版本定位

v2.2 是 v2.1.1（视觉统一版）之后的**功能补全版本**，聚焦统计分析模式 Phase 2-3 的完整交付，补齐"时间轴回放 + 图表下钻 + 报告生成 + 多格式导出"能力，同时优化全局加载体验。

### 2.2 核心问题

| 问题 | 现状 | 目标 |
|------|------|------|
| 看完带不走 | 分析结果只能看，不能导出 | 支持 Excel/PDF/PPT 一键导出 |
| 回不去 | 历史事件只能看静态数据 | 时间轴拖动回放态势变化 |
| 交不了差 | 汇报材料需手动截图拼凑 | 选模板→生成报告→导出PPT |
| 看不深 | 图表只有概览，无法下钻 | 点击数据点展开明细 |
| 体验生硬 | 模式切换无过渡 | 骨架屏加载动画 |

### 2.3 产品目标

1. 统计分析模式具备完整的"查→看→比→导"闭环能力
2. 支持一键导出 Excel/PDF/PPT 格式报告
3. 提升整体交互体验（加载态、筛选联动）

### 2.4 目标用户

| 用户角色 | 核心诉求 | 使用频率 |
|---------|---------|---------|
| 交通管理领导 | 看报告、要汇报材料 | 周/月 |
| 指挥中心值班员 | 复盘事件、对比策略效果 | 日 |
| 项目验收方 | 评估系统能力完整度 | 一次性 |

### 2.5 成功指标

- 统计分析模式功能完成率：68% → 100%
- 整体功能完成率：85% → 93%
- 导出报告可直接用于汇报（无需二次加工）

### 5.2 新增组件说明

**时间轴控制器（底部 80px）**
- 全宽横向时间轴滑块
- 左侧：播放/暂停按钮 + 速度选择器
- 中间：可拖动时间轴，标注事件节点（彩色圆点）
- 右侧：当前时间显示
- 仅在选中某个历史事件后激活

**报告管理面板（左栏下部）**
- Tab切换：模板 | 已生成
- 模板Tab：4种模板卡片（日报/周报/事件复盘/领导汇报）
- 已生成Tab：报告列表（时间倒序，点击预览）

**导出操作区（右栏下部）**
- 3个导出按钮（Excel/PDF/PPT）
- 按钮下方显示导出状态（生成中.../已完成）
- 导出范围跟随当前查询条件；若未设置筛选条件，默认导出近7天数据
- 数据量超过 500 条时，弹出确认提示："数据量较大（X条），导出可能需要较长时间，是否继续？"

**下钻明细面板（中央叠加层）**
- 点击图表数据点触发
- 半屏抽屉从右侧滑入
- 展示该时刻的详细指标、关联事件、策略执行情况
- 右上角关闭按钮

### 5.3 交互规则

| 场景 | 规则 |
|------|------|
| 下钻面板 vs 报告预览 | 互斥：打开一个自动关闭另一个 |
| 时间轴回放中切换视图Tab | 暂停回放，新视图从当前时间点渲染 |
| 时间轴未选中事件时 | 时间轴控制器置灰不可操作，显示提示"请先选择一个历史事件" |
| 导出范围 | 跟随当前查询条件；无筛选时默认近7天 |
| 导出中再次点击 | 按钮置灰，显示进度，禁止重复触发 |

### 5.4 异常场景处理

| 异常 | 处理方式 |
|------|---------|
| 导出失败 | Toast 提示"导出失败，请重试"；PPT失败时建议"可尝试导出PDF格式" |
| 报告生成失败 | 报告状态标记为"生成失败"，提供重试按钮 |
| 查询结果为空 | 中央区显示空状态插图 + "未找到匹配数据，请调整筛选条件" |
| 时间轴数据加载慢 | 显示时间轴骨架屏，加载完成后平滑过渡 |
| 图表截图失败（导出时） | 跳过该图表，在报告中显示"[图表加载失败]"占位，不阻断整体导出 |

### 5.5 导出主题

导出的 PDF/PPT 采用**浅色主题**（白底 + 深色文字），适合打印和投屏：
- 背景：#FFFFFF
- 标题色：#1A3A5C
- 正文色：#333333
- 强调色：#4DA6FF（与系统主色一致）
- 图表截图保持深色原样（作为视觉亮点嵌入白底报告中）

## 6. 技术方案

### 6.1 技术选型

| 能力 | 方案 | 理由 |
|------|------|------|
| Excel导出 | SheetJS (xlsx) | 纯前端生成，无需后端，社区成熟 |
| PDF导出 | html2canvas + jsPDF | 将DOM渲染为图片再转PDF，保留样式 |
| PPT生成 | PptxGenJS | 纯前端生成pptx，支持图表和图片嵌入 |
| 图表截图 | html2canvas | 导出时将Recharts图表截图嵌入报告 |
| 时间轴 | 自定义组件 + Framer Motion | 拖动交互 + 动画，项目已有Framer Motion |
| 骨架屏 | Tailwind animate-pulse | 项目已有Tailwind，零额外依赖 |

| 风险 | 等级 | 影响 | 缓解措施 |

## 11. 验收标准

| 场景 | 验收条件 |
|------|---------|
| 时间轴回放 | 选中事件后可拖动回放，图表随时间变化更新 |
| 图表下钻 | 点击任意图表数据点，右侧抽屉展示明细 |
| 报告生成 | 选择模板后 5 秒内生成报告，内容包含图表+AI结论 |
| Excel导出 | 点击后 3 秒内下载 .xlsx 文件，数据完整 |
| PDF导出 | 点击后 5 秒内下载 .pdf 文件，排版正确 |
| PPT导出 | 点击后 5 秒内下载 .pptx 文件，可直接投屏 |
| 骨架屏 | 模式切换时显示骨架屏，数据加载完成后平滑消失 |
| 筛选联动 | 修改筛选条件后图表 500ms 内自动更新 |
