## 八、风险与注意事项

### 技术风险
1. **TTS 兼容性**：Web Speech API 在不同浏览器表现不一致
   - 缓解：提供静音选项，播报失败时只显示文字

2. **性能影响**：虚拟人动画可能影响性能
   - 缓解：使用 CSS 动画而非 JS 动画，减少重绘

3. **告警频率**：告警过于频繁可能打扰用户
   - 缓解：10 分钟去重 + 用户可关闭

### 业务风险
1. **AI 预测准确性**：第一版全部基于 Mock 规则
   - 缓解：明确标注"模拟数据"，降低用户期望

2. **虚拟人形象接受度**：用户可能不喜欢卡通形象
   - 缓解：提供最小化选项，不强制展示
