# 虚拟人助手（主动播报）

> **所属模块**：总览模式
> **使用端**：PC 大屏
> **前置阅读**：`1_common.md`

## 目标

让总览模式具备“会说话”的表达能力，增强值班提醒和演示效果。

## 新增项

1. 虚拟人窗口
2. 主动播报能力
3. 语音状态动画
4. 静音控制
5. 最小化/展开

## 形态建议

第一版推荐：
- 静态头像 + 动态语音波形
- 位置：右下角
- 不遮挡海峡通行指数（需重新排布）

## 支持的播报场景

1. 系统启动播报
2. 异常触发播报
3. 整点播报
4. 模式切换建议播报

## 播报内容来源

- AI 态势面板结论
- 主动告警内容
- 当前关键指标摘要

## 页面变化

右下角新增浮动助手窗口：
- 头像
- 状态灯（待机/播报）
- 波形动画
- 静音按钮
- 展开按钮

## 交互流程

1. 页面进入后虚拟人默认待机
2. 符合播报条件时，进入 speaking 状态
3. 显示语音文本 + 波形
4. 播报结束后回到待机
5. 用户可静音或最小化

## 数据需求

- AI 结论文本
- 告警文本
- 当前摘要文本
- 播报场景配置

## 技术建议

- 第一版使用 Web Speech API
- 预留讯飞 TTS 接口能力
- 增加播报队列和中断机制

## 优先级

- P0

## 验收标准

- [ ] 右下角可显示虚拟人窗口
- [ ] 系统启动时可自动播报
- [ ] 异常时可自动播报
- [ ] 整点可自动播报
- [ ] 支持静音
- [ ] 支持最小化
