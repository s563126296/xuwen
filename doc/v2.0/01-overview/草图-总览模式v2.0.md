## 六、布局变化要点

### 主体布局（不变）
- 顶部导航
- AI 面板
- 左栏 320px / 中央地图 / 右栏 320px
- 底部图表区

### 主要变化
1. **顶部导航**：移除"应急"按钮（合并到指挥）
2. **AI 面板**：增强为态势判断中心
   - 折叠态新增：未来风险标签 + 可信度标签
   - 展开态新增：影响因子、趋势预判、相似案例、学习成果
3. **右栏底部新增**：虚拟人助手窗口（200×80px）
4. **全局浮层**：AI 主动告警（异常时弹出）
