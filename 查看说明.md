# 徐闻县智慧交通大屏

## 查看方式

### Windows 用户
双击 `启动.bat`，浏览器会自动打开。

### Mac 用户
1. 打开终端（Terminal）
2. 拖入 `启动.sh` 文件到终端窗口，按回车
3. 或者在终端中执行：
```bash
cd 解压后的目录
bash 启动.sh
```
浏览器会自动打开。

### 手动启动
如果以上方式不行，在 `dist` 目录下执行：
```bash
python3 -m http.server 8080
```
然后浏览器打开 http://localhost:8080

## 操作说明

- 顶部导航栏可切换 5 种模式：总览、港口、指挥、应急、AI分析
- 地图支持缩放和拖动
- 需要联网（地图使用高德地图 API）

## 分辨率

最佳显示分辨率：1920 x 1080（全屏浏览器）
