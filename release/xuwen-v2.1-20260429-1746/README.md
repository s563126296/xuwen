# 徐闻智慧交通大屏 v2.1 — 部署说明

## 快速启动

### 方式一：使用启动脚本（最简单，推荐）

**Windows 用户：**
双击 `启动系统.bat` 文件即可

**Mac/Linux 用户：**
双击 `启动系统.sh` 文件，或在终端运行：
```bash
./启动系统.sh
```

脚本会自动检测环境（Python 或 Node.js），选择可用的启动方式，并自动打开浏览器。

---

### 方式二：手动启动

如果启动脚本无法运行，可以手动启动：

**使用 Python（推荐）：**
```bash
cd xuwen-v2.1-20260429-1746
python3 -m http.server 8080
```
然后打开浏览器访问 http://localhost:8080

**使用 Node.js：**
```bash
cd xuwen-v2.1-20260429-1746
npx serve .
```
然后打开浏览器访问 http://localhost:3000

## 常见问题

**Q：地图区域显示空白怎么办？**
A：网络无法访问高德地图。检查网络连接，确保能访问 webapi.amap.com。

**Q：双击启动脚本没反应？**
A：请确保已安装 Python 或 Node.js。如果都没有，请先安装其中一个。

**Q：Mac 提示"无法打开，因为它来自身份不明的开发者"？**
A：右键点击 `启动系统.sh`，选择"打开"，然后点击"打开"按钮。
