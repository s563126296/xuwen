## 二、如何打开系统（3 种方式）

### 方式 1：使用 npx serve（最简单，推荐）

**步骤：**
1. 解压 `xuwen-v2.1-20260429.zip`
2. 打开终端（Mac）或命令提示符（Windows）
3. 进入解压后的目录：
   ```bash
   cd xuwen-v2.1-20260429-1703
   ```
4. 运行命令：
   ```bash
   npx serve .
   ```
5. 看到提示后，打开浏览器访问 `http://localhost:3000`

**优点：** 无需安装任何软件，npx 会自动下载 serve 工具

---

### 方式 2：使用 Python（Mac/Linux 自带）

**步骤：**
1. 解压 `xuwen-v2.1-20260429.zip`
2. 打开终端
3. 进入解压后的目录：
   ```bash
   cd xuwen-v2.1-20260429-1703
   ```
4. 运行命令：
   ```bash
   python3 -m http.server 8080
   ```
5. 打开浏览器访问 `http://localhost:8080`

**优点：** Mac/Linux 系统自带 Python，无需安装

---

### 方式 3：双击 index.html（不推荐）

**步骤：**
1. 解压 `xuwen-v2.1-20260429.zip`
2. 双击 `index.html` 文件

**缺点：** 
- 浏览器会以 `file://` 协议打开，可能导致部分功能无法正常工作
- 高德地图可能无法加载
- 不推荐用于正式演示

## 三、系统要求

### 浏览器要求
- **推荐：** Chrome 90+ 或 Edge 90+
- **支持：** Firefox 90+、Safari 14+
- **不支持：** IE 11 及以下

### 网络要求
- **必须联网**：系统需要加载高德地图（webapi.amap.com）
- 如果无法访问高德地图，地图区域会显示空白

### 硬件要求
- **分辨率：** 1920×1080 或以上（推荐大屏或投影仪）
- **内存：** 建议 8GB 以上
- **显卡：** 支持 WebGL（用于地图渲染）

## 五、演示流程（20 分钟）

### 开场（1 分钟）
- 一句话定位：AI 驱动的交通指挥决策系统
- 四大痛点：看不清、想不到、盯不住、学不会

### 故事演示（12 分钟）

#### 第一幕：AI 提前 30 分钟预警（总览模式，3 分钟）
1. 系统默认显示"日常平稳"场景
2. 点击右上角"场景预设"按钮
3. 选择"节假日高峰"或"港口积压"场景
4. 观察 AI 摘要面板颜色变化（绿色 → 黄色 → 橙色）
5. 观察告警浮层弹出
6. 点击"立即处置"进入指挥模式

#### 第二幕：AI 参谋长给方案（指挥模式 — 选策略，2 分钟）
1. 观察右侧面板的 AI 推荐策略卡片
2. 重点展示：
   - 推荐理由（为什么推荐这个策略）
   - 历史依据（30 次执行、87% 成功率、85% 匹配度）
   - 预期效果（资源 10 分钟到位，1 小时后拥堵指数从 6.8 降至 5.2）
   - 所需资源（交警 2 人 ✓、诱导屏 1 块 ✓）
3. 展开"备选策略"，查看完整信息
4. 点击"自定义策略"，展示策略库选择和手动新增功能

#### 第三幕：地图实时联动（指挥模式 — 执行中，4 分钟）
1. 点击"执行此方案"按钮
2. 观察地图变化：
   - **立即**：S376 分流路线从淡色虚线变成亮色动态流线
   - **3 秒后**：王警官、李警官的图标开始脉冲发光
   - **5 秒后**：视频窗口自动弹出，切换到 S376 路口摄像头
   - **6 秒后**：诱导屏图标闪烁
   - **15 秒后**：拥堵热力图颜色变化（红色 → 橙色）
3. 观察底部时间轴更新（显示执行进度和关键信息）
4. 观察左下角偏差监控面板（预期 vs 实际曲线）
5. 点击地图左上角图层筛选按钮，演示图层开关功能

#### 第四幕：AI 盯执行、发现偏差（指挥模式 — 偏差监控，1 分钟）
1. 观察左下角偏差监控面板
2. 查看 AI 归因分析（环境变化 50%、执行问题 30%、策略问题 20%）
3. 在聊天窗口输入模拟现场消息（可选）

#### 第五幕：复盘学习（指挥模式 — 执行完毕，1 分钟）
1. 等待策略执行完毕（或手动切换到完成状态）
2. 观察右侧面板切换到"执行完毕"状态
3. 查看执行结果摘要（拥堵指数 6.8 → 2.9，达成率 135%）
4. 查看 AI 学习摘要（+限高因子，准确率 82% → 86%）
5. 点击"查看复盘报告"，展示 5 部分结构化报告

#### 第六幕：策略实验室（AI 策略模式，2 分钟）
1. 点击顶部"AI 策略"按钮
2. 在策略模拟器中选择 1-2 个策略
3. 调整参数（天气改为大雾、货车比例调到 50%）
4. 点击"运行模拟"，观察曲线图
5. 切换到"进化记录" Tab，查看 AI 模型版本历史（v1.0 → v1.4，准确率 72% → 86%）

### 技术亮点（5 分钟）
- 30 分钟前瞻预测
- 11 因素归因引擎
- 策略-资源绑定
- 地图策略联动（4 阶段渐进展示）

### 总结（2 分钟）

- 三个模式闭环：监控 → 决策 → 执行 → 复盘 → 学习

## 六、常见问题

### Q1：地图区域显示空白怎么办？
**A：** 网络无法访问高德地图。检查网络连接，确保能访问 webapi.amap.com。

### Q2：系统加载很慢怎么办？
**A：** 首次加载需要下载地图资源，建议提前 5-10 分钟打开系统。

### Q3：场景切换后数据没有变化？
**A：** 等待 1-2 秒，系统需要时间更新数据。如果还是没变化，刷新页面重试。

### Q4：视频窗口没有自动弹出？
**A：** 确保已经点击"执行此方案"按钮，等待 5 秒后视频会自动弹出。

### Q5：能否离线演示？
**A：** 不能。系统依赖高德地图，必须联网使用。

### Q6：能否在手机或平板上演示？
**A：** 不推荐。系统为大屏设计（1920×1080），在小屏幕上体验不佳。
